# 🎭 Sentiment Analysis NLP Project
An end-to-end Natural Language Processing (NLP) pipeline that classifies text reviews into Positive, Negative, or Neutral sentiments.

## 🚀 Project Overview
This project utilizes **Naive Bayes classification** and **TF-IDF Vectorization** to understand the emotional tone behind customer reviews.

## 📊 Key Features
**Text Preprocessing:** Automated cleaning including lowercasing, tokenization, and stop-word removal.
**Vectorization:** Implements TF-IDF (Term Frequency-Inverse Document Frequency) to convert text into machine-readable numerical data.
**Performance Tracking:** Includes a detailed Classification Report (Precision, Recall, F1-Score) and a Confusion Matrix visualization[cite: 9].
**Live Demo:** Includes a real-time prediction function to test custom phrases[cite: 9].

## 🛠️ Tech Stack
* **Language:** Python
* **NLP Tools:** NLTK (Natural Language Toolkit) [cite: 4]
* **ML Libraries:** Scikit-learn, Pandas, NumPy [cite: 4]

## 📂 How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the script: `python sentiment_model.py`.
